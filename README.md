# 🐍 Python Fun Mini Projects

A collection of beginner-friendly Python mini projects designed to strengthen core Python programming concepts through hands-on practice. Each project focuses on improving problem-solving skills, logical thinking, and coding fundamentals in a fun and interactive way.

## 📂 Projects Included

* ✅ Number Guessing Game
* ✅ Word Guessing Game
* ✅ Hangman Game
* 🚧 More exciting mini projects coming soon!

## 🎯 Goal

This repository aims to help beginners learn Python by building simple, practical, and enjoyable projects while developing confidence in programming.
